Possuímos tecnologias diversas além das mostradas abaixo,
porém essas são as principais delas.

Java

Uma das mais usadas linguagens de programação, Java é primordialmente usado para desenvolvimento back-end incluindo projetos de grande escala (big-data) e desenvolvimento Android. Java é geralmente usado para programação desktop, mobile e games.

React.Js

O React é uma das nossas principais tecnologias, não só porque é o framework líder em desenvolvimento web nos dias de hoje, mas também sua presença nesse contexto. O fato de ser bem desenvolvido com uma biblioteca testada.

Python

Com uma sintaxe simples e uma grande flexibilidade, Python pode ser hoje em dia uma das mais produtivas linguagens para se trabalhar. De forma geral, é nossa linguegem de escolha para desenvolvimento de back-end de nossas aplicações, possibilitando rápidas adaptações para nossos clientes.

Node.js

Node.js é primordialmente usado para non-blocking, é usado tradicionalmente para desenvolvimento web e APIs back-end.

AWS

A AWS (Amazon Web Services) é um serviço de computação em nuvem desenvolvido pela Amazon. Ela oferece mais de 200 serviços completos de data centers por todo o mundo. Com isso, acaba trazendo mais recursos do que outros provedores de nuvem, segundo a própria empresa.

Hyperledger

O Hyperledger é um projeto abrangente de blockchains de código aberto e ferramentas relacionadas, iniciado em dezembro de 2015 pela Linux Foundation e recebeu contribuições da IBM, Intel e SAP Ariba, para apoiar o desenvolvimento colaborativo de sistemas distribuídos baseados em blockchain.
