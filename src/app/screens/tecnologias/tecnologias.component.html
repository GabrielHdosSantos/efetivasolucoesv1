Possuímos tecnologias diversas além das mostradas abaixo,
porém essas são as principais delas.

Java

Uma das mais usadas linguagens de programação, Java é primordialmente usado para desenvolvimento back-end incluindo projetos de grande escala (big-data) e desenvolvimento Android. Java é geralmente usado para programação desktop, mobile e games.

Angular

O Angular é uma das nossas principais tecnologias, não só porque é o framework líder em desenvolvimento web nos dias de hoje, mas também sua presença nesse contexto. O fato de ser bem desenvolvido com uma biblioteca testada.

Node.js

Node.js é primordialmente usado para non-blocking, é usado tradicionalmente para desenvolvimento web e APIs back-end.

AWS

A AWS (Amazon Web Services) é um serviço de computação em nuvem desenvolvido pela Amazon. Ela oferece mais de 200 serviços completos de data centers por todo o mundo. Com isso, acaba trazendo mais recursos do que outros provedores de nuvem, segundo a própria empresa.

Terraform

Terraform é uma ferramenta de software de código aberto, infraestrutura como configuração. Os usuários definem e fornecem infraestrutura de data center usando JSON.

Docker

Docker é um conjunto de produtos de plataforma como serviço que usam virtualização de nível de sistema operacional para entregar software em pacotes chamados contêineres. Os contêineres são isolados uns dos outros e agrupam seus próprios softwares, bibliotecas e arquivos de configuração.
